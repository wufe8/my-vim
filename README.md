## 使用

使用`install.bat`或`install.sh`一键安装 如果已经配置 会进行警告 并备份原配置 注意转移至其他安全位置

## 文件结构
```
myvim
|
+-- .config //用户配置
|    +-- nvim
|        +-- init.vim //nvim配置
+-- .vimrc //vim配置
+-- install.bat //windows下一键配置vim&neovim 需要git
+-- README.md //本文件
+-- vanilla.vimrc //无插件配置
+-- vim.md //笔记
```
